# Writing Tips

Good writing is about clarity and brevty. Every sentence should serve a purpose, and unneccesary words should be removed.

## Common Mistakes

Here are some things to watch out for:

- **Passive voice**: The ball was throwed by the boy.
- **Wordiness**: In order to acheive success, you must work hard.
- **Spelling**: The recieved package arrived on Wenesday.

> "The difference between the almost right word and the right word is the difference between the lightning bug and the lightning." — Mark Twain

## Example Code

Here is a Python function that counts words:

```python
def count_words(text):
    """Return the number of words in text."""
    return len(text.split())
```

Use `count_words("hello world")` to test it.
